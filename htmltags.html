HTML Tags Reference

1. html

Purpose: Root element of a webpage.

When to Use: Used in every HTML document.

<html></html>

2. head

Purpose: Contains page information.

When to Use: Before the body tag.

<head></head>

3. title

Purpose: Page title.

When to Use: Inside head.

<title>My Website</title>

4. body

Purpose: Visible page content.

When to Use: Main webpage area.

<body></body>

5. header

Purpose: Page header.

When to Use: Top section of page.

<header></header>

6. main

Purpose: Main content.

When to Use: Central page content.

<main></main>

7. section

Purpose: Content section.

When to Use: Divide content into parts.

<section></section>

8. article

Purpose: Independent content.

When to Use: Articles, blog posts, topics.

<article></article>

9. footer

Purpose: Footer section.

When to Use: End of webpage.

<footer></footer>

10. h1

Purpose: Main heading.

When to Use: Page title.

<h1>Internet</h1>

11. h2

Purpose: Section heading.

<h2>Heading</h2>

12. h3

Purpose: Sub heading.

<h3>Heading</h3>

13. p

Purpose: Paragraph.

<p>Hello World</p>

14. img

Purpose: Display image.

<img src="image.jpg">

15. table

Purpose: Create table.

<table></table>

16. tr

Purpose: Table row.

<tr></tr>

17. td

Purpose: Table data cell.

<td>Data</td>

18. ul

Purpose: Unordered list.

<ul></ul>

19. li

Purpose: List item.

<li>Item</li>

20. div

Purpose: Container.

<div></div>

21. video

Purpose: Display video.

<video></video>

22. iframe

Purpose: Embed maps, PDFs, YouTube.

<iframe></iframe>

23. link

Purpose: Connect CSS file.

<link rel="stylesheet" href="style.css">
